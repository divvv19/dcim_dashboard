# Plan 02-03: Historical Power Charting

## Objective
Embed responsive SVGs inside the frontend blocks that graph historical efficiency limits using the `recharts` package.

## Requirements
- Npm package `recharts` must be installed.
- The `request_history` socket handler in `server.js` must query the `facility` metric bucket alongside `environment`.
- The `PDUView` and `UPSView` panels must utilize `<ResponsiveContainer>` wrappers attached to a `<LineChart>` component processing `pue` trends over time.

## Tasks
1. Run `npm install recharts`.
2. Modify `server.js` -> `request_history` to `filter(fn: (r) => r._measurement == "environment" or r._measurement == "facility")`.
3. Import `LineChart`, `Line`, `XAxis`, `YAxis`, `Tooltip`, `ResponsiveContainer` into `DCIM.jsx`.
4. Wrap `chartData` using React's `useMemo` to group arrays by timestamp.
5. Inject the `<ResponsiveContainer>` underneath the PDU and UPS blocks to render the line graphs visually cleanly.

## Verification
- Wait 5 minutes for the `server.js` mock polling to log time-series data.
- The graphs should draw continuous lines tracking PUE flux inside `PDUView`.

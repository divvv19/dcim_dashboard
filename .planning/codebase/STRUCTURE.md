# STRUCTURE

## Top-Level Layout
- `.planning/` - Planning artifacts (this mapping). See `.planning/codebase/`.
- `dist/` - Built frontend output (Vite).
- `public/` - Static assets served by Vite.
- `server/` - Backend service (Express + Socket.IO).
- `src/` - Frontend source (React + Tailwind).
- `index.html` - Vite HTML shell.
- `package.json` - Frontend dependencies and scripts.
- `vite.config.js`, `tailwind.config.js`, `postcss.config.js`, `eslint.config.js` - Tooling configs.

## Frontend (`src/`)
- `src/main.jsx` - React entry point.
- `src/App.jsx` - App root component.
- `src/DCIM.jsx` - Main dashboard UI, component library, layouts.
- `src/hooks/useRealtimeData.js` - Socket.IO hook.
- `src/index.css`, `src/App.css` - Base styles.
- `src/assets/` - Static assets (if any).

## Backend (`server/`)
- `server/server.js` - Express + Socket.IO server, mock data simulator.
- `server/package.json` - Backend dependencies and scripts.

## Build Artifacts
- `dist/` - Generated by Vite build.
- `node_modules/` - Present at root and in `server/` (local installs).

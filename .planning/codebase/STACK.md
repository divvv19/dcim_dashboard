# Tech Stack

## Frontend
- **Framework**: React 19 (via Vite)
- **Styling**: Tailwind CSS v4, PostCSS
- **Icons**: lucide-react
- **Real-time**: socket.io-client
- **Build Tool**: Vite

## Backend
- **Runtime**: Node.js
- **Framework**: Express
- **Real-time**: Socket.IO
- **Protocols**: Modbus (modbus-serial), SNMP (net-snmp)

## Tooling
- ESLint for static analysis
- Nodemon for backend development
